# 간단한 게시판 프로젝트 
Maven + Spring MVC 를 이용한 간단한 게시판입니다. 

## 환경 설정
  * OS : Mac OS Catalina
  
  * IDE : IntelliJ IDEA 11.05
  
  * Version
    * Java : 1.8
    * Maven : Apache Maven 3.6.3
    * Spring : 4.3.18.RELEASE
    * Servlet : 3.1.0
    * MyBatis
      * MyBatis : 3.4.6
      * MyBatis-spring : 1.3.2
  
  * Database
    * MySQL : 8.0.18
  
  * WAS
    * Tomcat : Apache Tomcat 8.5.45
